# UniKostyl
Расширенная версия проекта [OpenKostyl](https://github.com/WildBoar228/OpenKostyl).
Предоставляет интерфейс для цветовой калибровки камеры, т.е. подбора цветового диапазона в формате LAB для некоторых объектов интереса.

В отличие от OpenKostyl, совместим с любой камерой (принимает кадр в формате numpy-массива). Также добавлен алгоритм удаления помех
на основе метода динамического программирования: он сужает цветовой диапазон, убрав лишние выделенные цвета, при этом оставив
диапазон максимального объёма в цветовом пространстве. Ввиду вычислительной сложности алгоритм вызывается из написанной на C++
библиотеки noise.dll, не рекомендуется применять "-" на очень больших диапазонах.

### Интерфейс `unikostyl.py`:
* `main_loop_frame(image_pixels: np.array)`: принимает в качестве аргумента кадр с камеры, обеспечивает работу
с пользовательским интерфейсом.
* `save_to_camera()`: сохраняет введённые диапазоны в текстовый файл.

## Примеры использования
* `OpenKostyl.py`: переработанная версия для OpenMV.
* `cv_kostyl.py`: использует веб-камеру устройства (для тестирования).
* `rpicam_kostyl.py`: экспериментальная версия для Raspberry Pi Camera.
